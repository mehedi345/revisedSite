What is tag?

A tag is an element inserted into a document or file that changes the look of content or performs an action.Tag has two parts,starting and closing.Whole website are wrapped on html tag.

What is attribute?

Attrubute is defined additional information about tag such as href,src,title,type etc.This attribute tells exact information about tags.Example:href attr.. tell exact path of an anchor tag

What is semantic element?

Semantic HTML elements are those that clearly describe their meaning in a human- and machine-readable way.
